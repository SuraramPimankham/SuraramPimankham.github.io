# work — Mini projects

## CRUD (ทดลองได้ทันทีบน GitHub)

**เปิดหน้านี้:** [work/index.html](./index.html)  
หรือบน GitHub Pages: `https://<username>.github.io/work/`

กดปุ่ม **เปิดทดลอง Node.js / .NET** ได้เลย — ไม่ต้องรัน `npm` หรือ `dotnet` (โหมดทดลองเก็บข้อมูลในเบราว์เซอร์)

## CRUD backends (MVC)

| โปรเจกต์ | ทดลอง (ไม่ต้องรัน server) | API จริง (optional) |
|---------|---------------------------|---------------------|
| Node.js (MVC) | [crud-nodejs/index.html](./crud-nodejs/index.html) | ดับเบิลคลิก `crud-nodejs/run.bat` |
| .NET (ASP.NET Core MVC) | [crud-dotnet/index.html](./crud-dotnet/index.html) | ดับเบิลคลิก `crud-dotnet/run.bat` |

เมื่อรัน `run.bat` แล้ว หน้าเว็บจะเชื่อม API อัตโนมัติ (แสดงป้าย “เชื่อมต่อเซิร์ฟเวอร์แล้ว”)

### ความต้องการ

- **Node:** Node.js 18+
- **.NET:** .NET SDK 10+ (`dotnet --version`)
