Mini CRUD

กดปุ่มด้านล่าง = โหมดทดลอง (localStorage, ไม่มี Network). ถ้าต้องการ Backend จริง ให้รัน run.bat แล้วเปิด localhost

Node.js

Backend แบบ MVC · http module (ไม่ใช้ Express)

เปิดทดลอง Node.js
รัน server — PowerShell (คัดลอกแล้ววางใน terminal) 
cd work\crud-nodejs; node server.js

เปิด → http://localhost:3000

? Debug Frontend Frontend (เบราว์เซอร์)
  1. รัน run.bat แล้วเปิด http://localhost:3000
  2. ต้องเห็นป้าย Backend จริง (ไม่ใช่โหมดทดลอง)
  3. F12 → Sources → เปิด crud-shared/crud-ui.js
  4. ใส่ breakpoint ที่ createItem (~175) หรือ updateItem (~200)
  5. กดบันทึก/อัปเดต → ดู payload ใน Variables
  6. ดู Network → Filter Fetch/XHR → POST/PUT /items
 ? Debug Backend Backend (Node.js MVC)
  1. รัน run.bat หรือ node server.js
  2. ใน Cursor เปิด controllers/itemsController.js
  3. ใส่ breakpoint ใน create / update
  4. Terminal: node inspect server.js แล้ว Chrome → chrome://inspect
  5. หรือใช้ JavaScript Debug Terminal แล้วรัน node server.js

.NET

Backend แบบ ASP.NET Core MVC · API ที่ /items

เปิดทดลอง .NET
รัน server — PowerShell (คัดลอกแล้ววางใน terminal) 
cd work\crud-dotnet; dotnet run

เปิด → http://localhost:5000

? Debug Frontend Frontend (เบราว์เซอร์)
  1. เปิด http://localhost:5000 (หลังรัน server แล้ว)
  2. ต้องเห็นป้าย Backend จริง
  3. F12 → Sources → crud-shared/crud-ui.js
  4. Breakpoint ที่ form submit (~277) หรือ updateItem (~200)
  5. ดู Console: [CRUD] Network PUT ...
  6. Network → PUT /items/{id} → แท็บ Payload
 ? Debug Backend Backend (.NET MVC)
  1. อย่าใช้ run.bat ตอน debug — ใช้ F5 เท่านั้น
  2. เลือก config CRUD .NET (Debug)
  3. Breakpoint ที่ ItemsController.cs → Create (~30) / Update (~43)
  4. รอ Now listening on: http://localhost:5000
  5. เปิด localhost:5000 → แก้ไข → อัปเดต → ดู input.Name
  6. Extension: anysphere.csharp (C# Dev Kit ใช้ใน Cursor ไม่ได้)
รัน server: เปิด terminal ที่โฟลเดอร์ repo แล้วกด Copy ด้านบน (หรือดับเบิลคลิก run.bat ในโฟลเดอร์โปรเจกต์)
Debug .NET: อย่ารันคำสั่งด้านบน — ใช้ F5 → CRUD .NET (Debug) ใน Cursor แทน

Breakpoint ไม่หยุด? เช็คว่า (1) เปิดจาก localhost ไม่ใช่ GitHub Pages (2) ป้ายเป็น Backend จริง (3) .NET debug ต้องใช้ F5 ไม่ใช่ dotnet run
